## Features

### Free Features
- **System Scan** - Deep scan to identify privacy risks and digital traces
- **File Shredder** - Secure file deletion with multi-pass overwriting
- **Timestamp Modifier** - Modify file timestamps to protect activity timeline
- **Anti-Analysis Detection** - Detect VMs, debuggers, and analysis environments

### Pro Features
- **Memory Cleanup** - Clear sensitive data from system memory
- **Network Cleanup** - Erase network traces and connection history
- **System Logs** - Secure cleanup of system and application logs
- **Disk Encryption** - AES-256 encryption for sensitive files
