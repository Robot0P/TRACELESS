常见问题

macOS 安装常见问题解决方案

"应用已损坏，无法打开"

此错误是因为应用没有 Apple 开发者证书签名。解决方法：

xattr -cr /Applications/Traceless.app

"无法打开，因为无法验证开发者"

macOS 的 Gatekeeper 阻止了应用。您可以通过以下方式允许：

  1. 打开 系统偏好设置 → 安全性与隐私
  2. 点击被阻止应用旁边的 仍要打开 按钮
  3. 或者右键点击应用并选择 打开

临时禁用 Gatekeeper

如果上述方法无效，您可以临时禁用 Gatekeeper：

sudo spctl --master-disable

⚠️ 安装完成后请记得重新启用：sudo spctl --master-enable

完全磁盘访问权限（可选）

为了完整功能，请授予完全磁盘访问权限：

  1. 打开 系统偏好设置 → 安全性与隐私 → 隐私
  2. 在左侧边栏选择 完全磁盘访问权限
  3. 点击锁图标并将 Traceless 添加到列表
